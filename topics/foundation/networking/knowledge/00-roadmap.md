# 网络基础学习 Roadmap（Foundation）

定位：本主题是支线能力，用于支撑 Agent 服务的通信、稳定性和排障能力。

## 1. 服务的主线场景

- `topics/system-design`：高可用服务、限流、重试、超时、熔断设计。
- `topics/agent`：工具调用链路稳定性与失败恢复。
- `topics/rag`：检索服务和模型服务的链路性能优化。

## 2. 学习范围

- 协议基础：TCP/UDP、HTTP/HTTPS、DNS、TLS。
- 服务治理：超时、重试、退避、熔断、负载均衡。
- 性能与稳定：连接池、队头阻塞、背压、长连接策略。
- 排障方法：抓包、日志关联、延迟分解、错误分类。

## 3. 产出要求

- `knowledge/`：网络与服务治理知识文档。
- `examples/`：最小客户端/服务端、超时重试示例。
- `qa/`：网络高频问答（面试和实战排障）。

## 4. 第一周任务

- [ ] 画出一次 Agent 请求的网络链路图。
- [ ] 写一个带 timeout/retry 的调用示例。
- [ ] 总结“什么时候重试，什么时候降级”。

## 5. 验收标准

- 能定位常见网络层故障并给出排查路径。
- 能解释超时与重试的副作用和治理策略。
- 能把网络知识映射到 Agent 工程稳定性方案。
