# Python 基础学习 Roadmap（Foundation）

定位：本主题是支线能力，用于支撑主线 `agent/rag/system-design` 的工程实现。

## 1. 服务的主线场景

- `topics/agent`：快速搭建 Agent 原型、脚本化流程、数据处理。
- `topics/rag`：文档切分、检索流水线、评估脚本。
- `topics/system-design`：服务端模块开发与可观测性接入。

## 2. 学习范围

- 语言基础：函数、类、模块、异常、上下文管理。
- 工程能力：虚拟环境、依赖管理、包结构、日志与配置。
- 并发与性能：`asyncio`、多进程基础、I/O 与 CPU 场景区分。
- 数据处理：JSON/CSV、文本处理、正则与常见数据结构。

## 3. 产出要求

- `knowledge/`：每个小主题 1 份知识点文档。
- `examples/`：每个小主题 1 个可运行示例。
- `qa/`：每周整理 5-10 个高频问答。

## 4. 第一周任务

- [ ] 完成 Python 工程脚手架最小模板。
- [ ] 写一个异步 API 调用示例（含超时与重试）。
- [ ] 写一份“Python 在 Agent 项目中的角色”总结。

## 5. 验收标准

- 能独立写出可维护的 Python 小模块。
- 能解释何时用同步、异步、多进程。
- 能把 Python 技能映射到至少 2 个主线场景。
